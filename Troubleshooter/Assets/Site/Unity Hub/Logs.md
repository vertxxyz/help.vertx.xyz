## Unity Hub logs
To view the Unity Hub logs select the account dropdown in the top left of the Hub and navigate to **â–¾ | Troubleshooting | Open log folder**.
^^^

| Operating system | Unity Hub log location                        |
|------------------|-----------------------------------------------|
| Windows          | `%UserProfile%\AppData\Roaming\UnityHub\logs` |
| Mac              | `~/Library/Application support/UnityHub/logs` |
| Linux            | `~/.config/UnityHub/logs`                     |
^^^ You can see this table in the [documentation](https://docs.unity3d.com/Manual/LogFiles.html)
