# Editor logs
To view the Player log, open a [Console window](https://docs.unity3d.com/Manual/Console.html) (**Window | General | Console**) and select **â‹® | Open Editor Log** from the window menu. You can also navigate to the following folder:

^^^

| Operating system | Player log location                      |
|------------------|------------------------------------------|
| Windows          | `%LOCALAPPDATA%\Unity\Editor\Editor.log` |
| macOS            | `~/Library/Logs/Unity/Editor.log`        |
| Linux            | `~/.config/unity3d/Editor.log`           |
^^^ You can see this table in the [documentation](https://docs.unity3d.com/Manual/LogFiles.html)
